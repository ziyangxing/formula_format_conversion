# 高中力学公式

## 运动学
- $v = v_0 + at$
- $x = v_0 t + \frac{1}{2} a t^2$
- $v^2 - v_0^2 = 2ax$
- $\bar{v} = \frac{x}{t} = \frac{v_0 + v}{2}$

## 牛顿定律
- $F = ma$
- $G = mg$
- $f = \mu N$
- $F = kx$

## 功、功率、机械能
- $W = Fx \cos\theta$
- $P = \frac{W}{t} = Fv$
- $E_k = \frac{1}{2}mv^2$
- $E_p = mgh$
- $E_{k1} + E_{p1} = E_{k2} + E_{p2}$

## 圆周运动
- $v = \frac{2\pi r}{T}$
- $\omega = \frac{2\pi}{T}$
- $a_n = \frac{v^2}{r} = \omega^2 r$
- $F_n = m\frac{v^2}{r} = m\omega^2 r$

## 万有引力
- $F = G\frac{Mm}{r^2}$
